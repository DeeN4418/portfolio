<h4>Доставка "Магнита"</h4>
<p>В проекте анализировались реальные данные курьерской доставки, на основании которых была выработана метрика эффективности доставки, а также построена диаграмма эффективности, с помощью которой можно оценить успешность выхода на слот. Также была оценена реальность рекламного предложения, что можно зарабатывать 4000 руб. в день (на осень 2024 года).</p>  
</div>

<div>
<h4>Демо-приложение "Голос в маршрут"</h4>
<p>Демо-приложение, созданное для хакатона от 2GIS (4-5 октября 2025 г.). Приложение состоит из статичной фронтенд-страницы, которая записывает фразу, отправляет ее на бэкенд (FastAPI). Бэкенд использует API 2ГИС для геокодинга (преобразования названий в координаты) и построения маршрута, а затем возвращает список координат на фронтенд для отображения на карте. Для реализации приложения были задействованы API 2GIS и ИИ-агенты CrewAI. </p>  
</div>
<hr>
<div>
<h4>Скрипты для обработки экспериментальных данных</h4>
  <p>В этом проекты реализованы python-скрипты для обработки экспериментальных данных, полученных с характериографа. <br>
    - Скрипт list_files.py производит обработку нескольких файлов, содержащих данные продолжительных измерений эмиссионного тока при фиксированной амплитуде импульса напряжения. На выходе получается файл со средними величиами эмиссионного тока и стандартного отклонения тока для последующих расчетов коэффициента вариации и других целей. <br>
   - Скрипт treat_file.py, который производит обработку нескольких файлов, содержащих данные измеренных вольт-амперных характеристик (ВАХ). На выходе получается файл с ключевыми эмиссионными параметрами, используемыми в дальнейшей аналитической работе. <br>
    Оба скрипта значительно сокращают процесс обработки экспериментальных данных.</p> 
</div>

<h4>Анализ сопротивлений</h4>
  <p>В проекте представлен анализ сопротивлений тонких пленок, полученных в ходе технологического процесса.<br>
    Проделаны следующие процедуры:<br>
    1. Загружены и обработаны данные из xls файла. Проанализирована статистика сопротивлений.<br>
    2. Проведен А/А-тест образцов, сформированных в холодное и теплое время года.<br>
    3. Построены матрицы корреляций.<br>
    4. Произведена кластеризация образцов по величине сопротивления.</p> <br>
</div>
